## 資料夾結構

```
cwsoft-meeting-tracker/
├── minutes/        # 原始逐字稿（.txt），由雅婷提供
├── meetings/       # 整理後的標準會議紀錄（.md）
├── projects/       # 各專案狀態檔（.md）
├── CLAUDE.md       # Claude Code 操作 SOP
└── README.md       # 本檔案（進度看板）
```

## 使用流程

1. 將雅婷逐字稿（.txt）放入 `minutes/`
2. 執行 Claude Code，它會依照 `CLAUDE.md` 的 SOP 自動處理
3. 查看 `meetings/` 確認會議紀錄，查看 `projects/` 確認任務更新
4. 本 README 的狀態看板會同步更新
